# llm-nigeria-livestock-benchmark


Benchmarking LLM performance on Nigerian livestock management knowledge.

## What this is
A 420-question evaluation dataset testing geographic and cultural bias in LLMs, focused on Nigerian livestock management practices.

## Models evaluated
- Llama 3.1 8B (via Groq) — pilot complete
- Claude Sonnet, GPT-4o, Gemini 2.5 Pro, Llama 3.1 70B — in progress

## Why it matters
Most LLM benchmarks are Western-centric. This project investigates performance gaps for Global South agricultural contexts.

## Preliminary findings
Bias gradient observed across knowledge categories in Llama 3.1 8B pilot.

## How to run
Open the notebook in Google Colab and follow the instructions inside.
